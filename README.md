# Kole

## Getting Started

System Requirements:

1. [Node.js 18.7^](https://nodejs.org/en) or later.
2. [pnpm - package manager](https://pnpm.io/installation#using-npm) (recommended)

First, install dependencies:

```bash
pnpm install
# or
npm run install
```

Now, run the development server:

```bash
pnpm dev
# or
npm run dev
```
